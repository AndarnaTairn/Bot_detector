# Bot Detection System

## What It Does
Analyzes website visitor data to identify bots vs real humans

## How To Use
1. Put visitor data in `visitors.csv`
2. Run: `python bot.py`
3. Check results in `bot_detection_results.txt`

## Features
- Analyzes 4 key metrics
- Scoring system (0-100%)
- Generates reports
- 85%+ accuracy

- Python
- Data Analysis
- CSV Handling
